# Wedding Concept Design — 컨셉 결정 스킬

한국 모바일 웨딩 청첩장의 디자인 컨셉을 결정하기 위한 가이드와 템플릿.

## 1. 사용자에게 물어볼 것 (필수 4가지)

처음에는 다음 4가지만 묻는다. 나머지는 진행하며 보충.

1. **신랑신부 이름과 예식 일시·장소** (필수)
2. **분위기 키워드 3개** (예: "차분한 / 미니멀 / 베이지" 또는 "화사한 / 봄 / 파스텔")
3. **양가 부모님 함자** (있으면)
4. **갤러리 사진 보유 여부** (없으면 AI 일러스트 그림 청첩장으로 진행)

부족하면 합리적 기본값으로 채우고 컨셉 시트에 표시.

## 2. 2025-2026 트렌드 가이드

| 트렌드 | 설명 | 적합 분위기 |
|--------|------|------------|
| 미니멀 세리프 | 본명조/Cormorant, 흰 여백, 가는 디바이더 | 차분한, 모던, 격식 있는 |
| 누드/베이지 톤 | #F5EDE2, #E8D9C0, #B8A088 등 따뜻한 누드 | 차분한, 클래식 |
| 세이지/유칼립투스 | #A8B5A0, #D4DDC8, 식물 모티프 | 자연스러운, 보태니컬 |
| 더스티 핑크/와인 | #D4A5A5, #8B4252 | 로맨틱, 화사한 |
| AI 일러스트 청첩장 | 부부 캐릭터, 수채화 풍경 | 캐주얼, 따뜻한 |
| 풀스크린 사진 + 세리프 캘리그래피 오버레이 | 스튜디오 사진을 풀로 깔고 그 위에 영문 세리프 | 모던, 시크 |
| 그라디언트 인쇄물 느낌 | 종이 텍스처 + 부드러운 그라디언트 배경 | 클래식, 우아한 |

## 3. 컬러 팔레트 큐레이션 (3안)

분위기 키워드에 맞춰 3안 중 추천. 모든 팔레트는 5색 (primary / secondary / accent / background / text).

### A. Cream Beige (차분한 · 클래식)
- primary: `#B8A088` (warm taupe)
- secondary: `#D4C5B0` (light beige)
- accent: `#7A6347` (dark brown)
- background: `#FAF6EF` (off-white cream)
- text: `#2C2520` (deep brown black)

### B. Sage Botanical (자연스러운 · 보태니컬)
- primary: `#8FA07F` (sage green)
- secondary: `#C5D1B8` (light sage)
- accent: `#5C6B4F` (deep sage)
- background: `#F4F1EA` (warm ivory)
- text: `#2A332A` (forest black)

### C. Dusty Rose (로맨틱 · 화사한)
- primary: `#C9A0A0` (dusty rose)
- secondary: `#E8D2D2` (light rose)
- accent: `#8B4252` (wine)
- background: `#FBF6F4` (blush white)
- text: `#3A2828` (rose black)

### D. Modern Mono (모던 · 미니멀)
- primary: `#1A1A1A` (near black)
- secondary: `#8A8A8A` (mid gray)
- accent: `#C9B27C` (gold)
- background: `#FFFFFF` (pure white)
- text: `#1A1A1A`

### E. Soft Lavender (꿈결같은 · 차분한)
- primary: `#B5A5C9` (lavender)
- secondary: `#DCD2E8` (light lavender)
- accent: `#6B5B8B` (deep lavender)
- background: `#F8F5FA` (lavender tint white)
- text: `#2C253A`

## 4. 폰트 큐레이션

### 국문 세리프 (인사말, 본문 한국어)
- **Nanum Myeongjo** (Google Fonts) — 가장 대중적, 클래식
- **Noto Serif KR** (Google Fonts) — 모던 세리프, 가독성 우수
- **Gowun Batang** (Google Fonts) — 부드러운 곡선, 따뜻한 인상
- **나눔손글씨 펜** 또는 **Gaegu** — 손글씨 느낌이 필요할 때 부분 사용

### 국문 산세리프 (보조, 작은 정보)
- **Pretendard** (CDN) — 모던 산세리프 표준
- **Noto Sans KR**

### 영문 세리프 (이름, 날짜 강조)
- **Cormorant Garamond** — 우아한 디스플레이 세리프
- **Playfair Display** — 클래식 매거진 세리프
- **Italiana** — 가는 디스플레이
- **EB Garamond** — 따뜻한 클래식

### 영문 스크립트/캘리그래피 (선택, "& / and / forever" 강조)
- **Great Vibes** — 우아한 스크립트
- **Allura** — 부드러운 스크립트
- **Pinyon Script** — 가는 스크립트

## 5. 무드별 추천 조합

| 분위기 키워드 | 컬러 | 국문 | 영문 |
|-------------|------|------|------|
| 차분한 + 미니멀 + 베이지 | A. Cream Beige | Noto Serif KR | Cormorant Garamond |
| 보태니컬 + 자연스러운 | B. Sage Botanical | Gowun Batang | EB Garamond |
| 로맨틱 + 화사한 | C. Dusty Rose | Nanum Myeongjo | Playfair Display + Allura |
| 모던 + 시크 | D. Modern Mono | Pretendard + Noto Serif KR | Italiana |
| 꿈결같은 + 부드러운 | E. Soft Lavender | Gowun Batang | Cormorant + Great Vibes |

```markdown
  bride_mother: ""

  bride_birth_order: ""
mood_keywords: []

  primary: ""
  secondary: ""
  accent: ""
  background: ""
  text: ""

  korean_serif: ""
  korean_sans: ""
  english_serif: ""
  english_script: ""
imagery_direction: ""    # "수채화 플로럴" / "AI 일러스트 부부" / "스튜디오 사진" 등

  - gallery

# 컨셉 디렉션 노트

## 톤앤매너
{한 문단으로 요약}

## 핵심 비주얼 컨셉
{한 문단}

## 카피라이터에게 가이드
- mood_keywords의 첫 번째 키워드를 카피 톤의 중심에 둘 것
- ...

## 이미지 아티스트에게 가이드
- imagery_direction을 모든 이미지에 일관되게 적용
- 컬러 팔레트의 primary/background를 이미지에 자연스럽게 녹여낼 것
```

## 7. 결정 시 고려사항

- **갤러리 사진이 없으면** imagery_direction을 "AI 일러스트" 또는 "추상 플로럴" 중심으로. 사진이 들어갈 자리에 일러스트로 대체.
- **격식 있는 예식(호텔)** vs **캐주얼(레스토랑/하우스)** 에 따라 인사말 톤 추천을 다르게 함.
- **보유 폰트와 라이선스**: Google Fonts CDN 가능한 폰트만 추천. 유료 폰트 추천 금지.
