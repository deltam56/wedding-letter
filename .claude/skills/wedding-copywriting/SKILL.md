# Wedding Copywriting — 청첩장 한국어 카피 가이드

청첩장 모든 텍스트의 톤·구조·관습을 정리한 가이드.

## 1. 인사말 (Greeting)

청첩장의 핵심. 컨셉 시트의 mood_keywords에 맞춰 어조를 결정한다.

### 어조 매핑
| mood_keywords 첫 키워드 | 어조 | 호흡 |
|----------------------|------|------|
| 차분한, 미니멀 | 짧은 문장, 여백 많음, 시적 | 4-6행 |
| 클래식, 격식 | 한자어 활용, 정중한 청유형 | 6-8행 |
| 화사한, 봄, 로맨틱 | 계절·자연 메타포 | 6-8행 |
| 모던, 시크 | 단정한 평서문, 영문 인용 가능 | 4-6행 |
| 따뜻한, 캐주얼 | 일상 대화체, 진심 강조 | 5-7행 |

### 분위기별 예시 풀

#### A. 차분한 · 미니멀
```
서로의 가장 가까운 자리에서
함께 걷기를 약속합니다.

귀한 걸음으로 축복해 주시면
큰 기쁨이 되겠습니다.
```

```
오랜 봄을 지나
서로에게 도착했습니다.

저희의 시작에
당신이 함께해 주시기를.
```

#### B. 클래식 · 격식
```
저희 두 사람,
사랑과 믿음으로 일생의 동반자가 되려 합니다.

귀한 걸음으로 축복해 주신다면
더없는 기쁨으로 새로운 길을 시작하겠습니다.
```

#### C. 화사한 · 로맨틱
```
꽃이 피는 봄날,
저희 두 사람이 부부가 됩니다.

따스한 마음으로 함께해 주신다면
오래도록 기억될 하루가 되겠습니다.
```

#### D. 모던 · 시크
```
오늘의 약속이
내일의 일상이 됩니다.

저희의 시작을 함께 기억해 주세요.
```

#### E. 따뜻한 · 캐주얼
```
오랜 시간 함께해 온 두 사람이
이제 부부라는 이름으로 살아가려 합니다.

가장 가까이에서 응원해 주신
당신께 가장 먼저 알립니다.

귀한 걸음 부탁드립니다.
```

### 작성 시 주의
- "결혼합니다" 단정 종결은 모던/캐주얼에서만. 클래식에서는 "혼인하고자 합니다", "백년가약을 맺고자 합니다".
- 클리셰 회피: "두 사람이 하나가 되는 자리", "사랑의 결실" 등은 가능한 피한다.
- 줄바꿈 위치는 한 호흡 단위로. 한 행 12-16자가 모바일 최적.

## 2. 부모님 함자 + 신랑신부 소개

### 표준 표기
```
{아버지 함자} · {어머니 함자} 의 {출생순서} {신랑 이름}
{아버지 함자} · {어머니 함자} 의 {출생순서} {신부 이름}
```

### 출생순서 표기
- 아들: 장남 / 차남 / 삼남 / 막내아들 / 외아들
- 딸: 장녀 / 차녀 / 삼녀 / 막내딸 / 외동딸
- 합쳐: 장남이자 외아들 (하나뿐인 아들)

### 별세 표기
- 故 ○○○ (한자 故 또는 한글 "고")

### 한 분만 계신 경우
- 어머니/아버지 한 분 단독: `{함자} 의 {출생순서} {이름}`

### 예시
```
김철수 · 이영희 의 장남 김민수
박상호 · 정수진 의 차녀 박지영
```

## 3. 예식 안내

### 일시 표기 패턴
- 한국식: `2026년 5월 17일 일요일 오후 1시 30분`
- 음력 병기(클래식 컨셉): `2026年 5月 17日 (음 4월 1일) 일요일 오후 1시 30분`
- 모던: `2026.05.17 SUN PM 1:30`

### 장소 표기
```
{장소명} {홀 이름}
{도로명주소} ({지번주소})
{호텔/예식장 층 정보}
```

## 4. 오시는 길 (Directions)

표준 4종: 지하철 / 버스 / 자가용·주차 / 셔틀버스(있을 시)

### 템플릿
```markdown
### 지하철
{호선} {역명} {몇 번 출구} 도보 {분} 분

### 버스
{정류장명} 하차
- 간선: 0000, 0000
- 지선: 0000

### 자가용 / 주차
지하 주차장 이용 가능 (예식 시간 기준 {시간} 무료)
네비게이션: "{장소명}" 또는 "{도로명주소}" 검색

### 셔틀버스 (있을 경우)
{출발지} → {장소} 운행
{시각}
```

## 5. 마음 전하실 곳 (Account Info)

### 도입 안내문 (예시)
```
참석이 어려우신 분들을 위해
마음 전하실 곳을 안내드립니다.

축하의 마음을 담아 주신다면
저희 두 사람에게 큰 힘이 되겠습니다.
```

### 짧은 버전 (모던 컨셉)
```
참석이 어려우시다면
마음 전하실 곳을 남겨두겠습니다.
```

### 계좌 표기
```
신랑측
{은행} {계좌번호} {예금주}
( 신랑 본인 / 혼주 )

신부측
{은행} {계좌번호} {예금주}
```

계좌번호 미입력 시 placeholder:
```
[○○은행 ○○○-○○○○-○○○○ {신랑 이름}]
```

## 6. RSVP 안내문

```
참석 의사를 미리 알려주시면
더욱 정성껏 준비하겠습니다.

소중한 시간 내어주시는 모든 분들께
미리 감사드립니다.
```

## 7. 방명록 안내

```
두 사람에게 전하고 싶은 말씀을
남겨주세요.

오래도록 간직하겠습니다.
```

placeholder 예시:
```
"축하의 메시지를 남겨주세요"
```

## 8. 카카오톡 공유 (kakao_share)

og:title (40자 이내):
```
{신랑 이름} ♡ {신부 이름} 결혼합니다
```

og:description (90자 이내):
```
{YYYY년 M월 D일 요일} 오후 H시 M분
{장소명} {홀 이름}
```

## 9. 출력 규칙

`_workspace/02_copy-writer_copy.md`로 저장. 각 섹션에 명시된 마크다운 헤더(`## hero_subtitle`, `## greeting` 등)를 정확히 사용해 frontend-developer가 파싱하기 쉽게 한다.

## 10. 체크리스트

작성 후 확인:
- [ ] 부모님 함자가 컨셉 시트와 정확히 일치
- [ ] 신랑신부 출생순서가 명시됨
- [ ] 예식 일시가 한국식 표기로 정확함
- [ ] 장소·주소가 컨셉 시트와 일치
- [ ] 계좌 정보 placeholder가 명확히 표시됨
- [ ] og:title, og:description이 카카오 글자수 제한 안에 들어감
- [ ] 인사말이 mood_keywords의 첫 키워드 어조에 부합
